## 引言

每一位英语学习者都曾经历过这样的困境：单词书背了又忘，语法规则学了不用，昂贵的课程上了却缺乏实战机会。我们花费大量时间"专门学习"，却发现真正使用时依然张不开嘴、看不懂文。

问题的根源在于——语言学习需要持续的、真实的、可理解的输入，而不是孤立的单词背诵。

著名语言学家 Stephen Krashen 提出的"可理解输入假说"（Comprehensible Input Theory）指出：人们只有接触到略高于当前水平（i+1）的语言材料时，才能真正内化语言能力。这解释了为什么很多人在英语国家生活几年后，即使不刻意学习，语言能力也会大幅提升——因为他们每天都在接触真实的、可理解的输入。

然而，对于大多数学习者而言，我们无法随时切换到英语环境。我们依然需要阅读中文新闻、使用中文网站、处理中文信息。

**FlowLingo** 正是为解决这一困境而生。

## 核心理念

FlowLingo 是一个 Chrome/Edge 浏览器扩展，它的核心理念是：**在你正常浏览网页的同时，让英语自然融入你的阅读体验**。

它不是让你"专门学习"，而是让你在浏览微博、阅读新闻、查看文章时，系统会自动识别并替换部分中文词汇或短语为对应的英语表达。通过这种方式，你接触到的每一处替换都是与当前语境紧密相关、难度适中的英语输入。

这种技术被称为"代码切换"（Code-Switching），原本指双语者在对话中自然切换语言的现象。FlowLingo 将这一概念应用于阅读场景，让中文网页变成一个个微型的双语学习环境。

更重要的是，FlowLingo 采用本地优先架构，核心逻辑在本地运行，网页解析在沙箱中完成，API 调用产生的文本也经过隐私脱敏处理——你在享受 AI 智能的同时，隐私安全得到充分保障。

## 四大核心功能

### AI 代码切换

FlowLingo 的核心功能是智能的词汇替换引擎。它不是简单的字典查询，而是借助大语言模型深度理解网页语境，识别最适合替换的词汇或短语。

对于初级用户，系统会聚焦于具体名词、常见动词和基础形容词——这些是你最需要建立基础认知的词汇。对于中级用户，替换范围扩展到抽象名词、短语动词和专业术语。对于高级用户，则会引入高级习语、精确形容词和学术词汇。

替换后的英语在语法和语义上与原中文语境完美融合。例如，一句"这家餐厅的味道很棒"可能被替换为"The 这家 restaurant 的 flavor 很不错"，既保持了句子的整体可读性，又植入了目标词汇。

系统还具备间隙冲突检测能力，确保相邻替换词之间保持适当间距，避免视觉上的拥挤感。此外，替换后的内容会经过智能缓存，相同内容在 1024 条目的 LRU 缓存中可实现秒级响应，避免重复调用 API。

### 自适应学习系统

FlowLingo 不仅是一个静态的替换工具，它更是一个能够"理解"你的学习进度并动态调整的智能系统。

系统为每个接触过的词汇维护一个"掌握度"（Mastery）指标，初始值为 0.5。当你悬停查看词汇卡片时，掌握度会略微降低，表示这个词汇还需要复习。当你主动标记"认识"时，掌握度会显著提高。当你标记"不认识"时，掌握度会明显降低。

基于这些实时的用户交互信号，系统会智能判断是否需要调整学习策略。当检测到高不识率（5 次交互中超过 50% 标记为"不认识"）、高恢复率（频繁点击恢复原句）或高暂停率（频繁暂停插件运行）时，系统会自动降低替换强度，并切换到更简单的呈现模式，确保学习体验始终保持积极和有效。

在开始使用前，系统会通过一个自适应的词汇量测试来评估你的英语水平，并据此推荐合适的 CEFR（欧洲语言共同参考框架）等级。这个等级将决定后续所有替换的词汇难度。

### 灵活的 LLM 集成

FlowLingo 并不绑定特定的 AI 服务商，而是采用开放式架构，兼容所有支持 OpenAI 标准接口的服务商。

你可以在设置中添加多个 API 节点，包括 DeepSeek、GPT-4o-mini、Groq、Moonshot、Ollama 等主流服务。系统支持两种调度策略：轮询（Round Robin）模式会将请求均匀分布到所有可用节点，实现负载均衡；优先级（Priority）模式则会优先使用列表顶部的节点，在节点失败时自动故障转移到下一个节点。

系统内置了主流 AI 服务商的预设配置，只需一键填充即可完成设置。同时支持全局和节点级别的速率限制管理，避免因超出 API 配额导致服务中断。

当某个节点调用失败时，系统会自动尝试下一个节点，确保在部分服务商出现问题的极端情况下，服务依然可用。

### 多模态交互体验

FlowLingo 提供了丰富的交互方式，让学习过程更加自然流畅。

在呈现模式上，你可以选择 English (中文)（英文为主，括号中文辅助，默认模式）、中文 (English)（中文为主，括号英文辅助）或 English Only（纯英文沉浸模式）。无论你处于什么学习阶段，都能找到最舒适的阅读方式。

在替换强度上，系统提供三档选择：初学探索（5-10% 密度）适合轻松浏览；进阶训练（10-20% 密度）适合日常学习；深度浸入（20-30% 密度）适合追求高强度训练的用户。

发音是语言学习的重要环节。FlowLingo 支持三种发音引擎：本地系统 TTS 完全离线可用；Google 翻译发音音质清晰；有道词典发音带有明显的英式发音风格。还可以开启悬停自动朗读功能，在鼠标悬停时自动播放发音。

## 快速上手指南

使用 FlowLingo 非常简单。

首先，通过"加载已解压的扩展程序"方式安装插件。打开浏览器扩展管理页面，开启开发者模式，点击加载已解压的扩展程序并选择项目目录。

安装完成后，点击插件图标进入选项页面。第一步是配置 LLM API Key，如果你还没有账号，可以选择 DeepSeek 等性价比较高的服务商。第二步是完成词汇量测试，系统会根据你的表现推荐合适的 CEFR 等级。

配置完成后，你就可以开始浏览网页了。打开任意中文资讯网站，系统会自动识别文章主体并开始注入英语接触点。

如果你不确定如何设置，可以参考以下推荐配置：入门用户可以选择 A1-B1 等级，搭配 en_cn 呈现模式和 low 替换强度；进阶用户可以选择 B1-B2 等级，搭配 cn_en 呈现模式和 medium 替换强度；高级用户可以选择 B2-C1 等级，搭配 en_only 呈现模式和 high 替换强度。

## 总结
FlowLingo 的核心价值在于它真正解决了"学以致用"的问题。它不需要你专门抽出时间，不需要改变已有的浏览习惯，只需要在日常的网络浏览中，英语就会自然地出现在你眼前。

智能的自适应系统确保你始终处于最佳的学习区间，不会因为太难而挫败，也不会因为太简单而无聊。开放的 LLM 架构让你可以根据成本和效果自由选择服务商。多模态的交互方式则让听说读写各个环节都有相应的支持。
